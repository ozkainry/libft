# **Libft - Your Very First Own Library**

## **Features**

Libft is designed to help you develop essential utilities that can be reused across multiple C projects. These utilities include functions for string manipulation, memory management, and more. It also contains the following additional components:

- **ft_printf**: A custom implementation of the `printf` function, allowing formatted output similar to the standard `printf`.
- **get_next_line**: A function to read a line from a file descriptor, useful for processing input line by line.

## **Usage**

Here are some useful commands to manage and use the Libft library:

- **`make`**: Compiles and creates the static library **libft.a**.
- **`make clean`**: Removes all object files (`.o`) generated during the compilation.
- **`make fclean`**: Removes both the object files and the **libft.a** library, effectively cleaning the entire build.
- **`make re`**: Recompiles everything from scratch (performs a `make fclean` followed by `make`).
